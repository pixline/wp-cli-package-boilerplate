# wp-cli boilerplate

Package template

## Setup

* Install [wp-cli](http://wp-cli.org)
* Add wp-cli package index:
```
composer config repositories.wp-cli composer http://wp-cli.org/package-index/
```

* Install command:
```
composer require pixline/wp-cli-boilerplate=dev-master
```
